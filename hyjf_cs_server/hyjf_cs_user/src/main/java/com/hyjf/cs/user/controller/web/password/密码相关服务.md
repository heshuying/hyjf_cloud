#密码相关服务能描述

##修改登陆密码
修改登陆密码<br/>
├── 校验参数<br/>
├──更新user表<br/>

##设置交易密码<br/>
设置交易密码<br/>
├── 校验参数（包含是否开户，是否已经设置过）<br/>
├── 设置请求银行参数（其中包含返回给前台的成功失败url）<br/>
├── 请求银行接口<br/>
├── 返回给前台银行返回值<br/>
├── 调用异步接口<br/>
